### Build & Run

#### 1. Clone the repository

```terminal
git clone https://github.com/Gug-er/BlackJackBot.git
```
#### 2. .venv
```terminal
python -m venv .venv
```
```terminal
.venv/Scripts/activate
```
#### 3. Requirements
```terminal
pip install -r requirements.txt
```
#### 4. .env
Create `.env` file in project directory
 
Add token for your bot from @BotFather
``` .env
TOKEN=[your token]
```
#### 5. Run
```terminal
python main.py
```
